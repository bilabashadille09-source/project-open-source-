# Project Description:

Our project aims to study the relationship between nutrition and liver diseases.
We collected scientific information about how diet affects liver health and identified eating habits that help prevent or worsen liver conditions.
The main goal of this project is to raise awareness about the importance of a balanced diet to maintain a healthy liver.


## Steps of Realization:

1. Research reliable scientific sources about nutrition and liver diseases.
2. Analyze how different foods affect liver function.
3. Write a short summary of the findings.
4. Create and complete the README file on GitHub.
5. Add a Project Board to track teamwork and progress.
6. Final review and submission of the project for evaluation.
